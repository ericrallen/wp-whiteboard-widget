== Description ==

Adds a custom dashboard widget that allows administrators to leave comments for one another.

Will allow custom user level access to be set up so admins can control who can see posts and allow users to specify access levels for their comments.

== Installation ==

1. Upload `ia-wp-whiteboard-widget` to your plug-in directory or install it from the Wordpress Plug-in Repository
2. Activate the plugin through the 'Plugins' menu in WordPress
